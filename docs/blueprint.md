# **App Name**: CampusConnect

## Core Features:

- Teacher Authentication: Secure login and registration for teachers with profile management.
- Student Authentication: Custom login form for students with mandatory initial password change.
- Class Management: Teachers can add, edit, and delete classes via the sidebar.
- Student Management: Display student list with behavior scores, bulk add functionality, and actions like password reset and chat.
- Project Distribution: Manage lesson quotas, view student preferences, and manually assign lessons.
- Risk Management: Add/remove risk factors and view student risks in a grid table.
- Info Form Management & Export: View student info form status and export detailed forms to Word documents with teacher info.

## Style Guidelines:

- Primary color: Deep indigo (#4B0082) to convey a sense of knowledge and trustworthiness.
- Background color: Light gray (#F0F0F0), for a clean and unobtrusive backdrop.
- Accent color: Muted purple (#800080) to highlight important interactive elements without overwhelming the primary color.
- Headline font: 'Space Grotesk', a sans-serif for a modern, computerized feel, paired with 'Inter' (sans-serif) for body text to ensure readability and neutrality.
- Lucide React icons: Consistent, minimalist icons for actions like Trash, Edit, UserPlus, FileText, ensuring clarity.
- Responsive sidebar: A collapsible sidebar (drawer on mobile) for class selection. Main area uses tabs for different functionalities within a class.
- Subtle transitions: Use subtle transitions for loading data, opening modals, and tab switching to enhance user experience.
